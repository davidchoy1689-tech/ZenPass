# Cold Call Script — 企業運動福利推銷

*適用對象：中小企 HR Manager / Admin Manager / 老闆（10-200人）*

---

## 電話前準備

### Target 公司清單（優先順序）

1. 🔥 **你認識嘅朋友/前同事公司** — 最好傾，第一轉打
2. 🔥 **你現有合作場地嘅租戶公司** — 已經有 connection
3. 🔥 **附近商業大廈（地鐵沿線）** — 地點近，方便員工上堂
4. 📍 LinkedIn 搜 「HR Manager + 香港」— 在職 full-time HR 優先

### 打電話前必 check

- [ ] 公司名點讀（唔好讀錯）
- [ ] 對方 title（叫 Miss Lee / 李生）
- [ ] 有冇 common connection 可以提
- [ ] 呢間公司規模（10-200人？有冇 wellness budget？）
- [ ] 準備好 demo link：`zenpass.hk/corporate-guide.html`

## 冷 Call 流程

### Step 1: 開場（5 秒）

> **喂【稱呼】，我係【你個名】呀，【共同朋友/公司】介紹我打嚟嘅。**

*如果冇介紹人：*

> **喂【稱呼】，我係 ZenPass 嘅【你個名】，打擾你 30 秒就得。你哋公司而家員工有冇 wellness 福利？**

---

### Step 2: Value Prop（15 秒）

**核心一句：**

> **我哋幫香港中小企俾員工運動福利，唔洗 HR 跟任何 admin。**
> ——員工手機 book 堂，系統自動扣公司 Credits，每月自動 reset，你喺 Dashboard 睇到晒。

*如果有人問「點解要搞呢啲？」：*

> **員工運動平均提升 12% 生產力、減 27% 病假。你每月 $200/人，回報最少 2 倍。**

---

### Step 3: 常見反對意見處理

#### 「我哋冇 budget / 唔需要」

> 明白。好多中小企開頭都覺得係額外開支，但其實係 investment。我同你計條數：一個員工每年病假成本大概 $5K-10K。我哋每月 $200/人（～2 堂運動），成本隨時 cover 到。
>
> 不如我 send 個 proposal 俾你睇？15 分鐘就明晒。

#### 「同事未必會用」

> 我哋設計咗 Credits 制 — 員工自己揀堂、預約、簽到，唔用每月歸零。公司俾咗錢，員工自然會想用盡。
>
> 而且開頭有免費試用 2 個月，試咗先知。

#### 「我哋已經有 gym pass / 其他福利」

> 明白。你而家用緊邊間？俾我參考下？⋯
>
> 其實我哋最大分別係：gym pass 係指定場地，員工嫌遠就唔去。我哋有 11 種運動（包括匹克球）+ 多區場地，員工自由度大好多。
>
> 而且你個 Dashboard 一頁睇到邊個有去邊個冇去，gym pass 俾唔到呢啲數據。

#### 「而家冇時間睇呢啲」

> 完全明白。咁我 send 個資料俾你 WhatsApp / Email，你自己得閒睇。
>
> 最快 Demo 係 15 分鐘 online，你揀時間。
>
> *（直接 send calendar link 或問時間）*

#### 「要問老闆 / 要同 management 傾」

> 明。不如我 send 個 1 頁 summary 俾你，你可以 forward 俾老闆睇。cover 晒重點。
>
> 你老闆叫咩名？我幫你整個 personalized 版本。

---

### Step 4: 收線 — 一定要拿下一步

**成功情況（對方有興趣）：**

> 唔該晒。我而家 send 個 proposal 俾你。我下星期幾日打嚟跟進？
>
> 定係你想約 15 分鐘 online Demo？我 send calendar link 俾你。

**中立情況（對方未決定）：**

> 冇問題。我 send 個 1 頁 summary + Demo link 俾你。
>
> **星期四或者星期五再打嚟同你傾幾分鐘，可以嗎？**

**負面情況（真係冇興趣）：**

> 明白。咁我有新嘢或者 promotion 嘅時候再 contact 你，唔打擾了。多謝你嘅時間。

---

  - 跟進日期

## 實戰小技巧

### 語氣

- **唔好背稿** — 聽落自然就得
- **快唔過對方** — 佢一聽就知你係 sales 會自動掛線
- **停一停** — 講完一句之後停 1 秒，俾對方消化
- **用問題代替陳述** — 「你哋有冇 wellness 福利？」比「我哋可以幫你」好

### 關鍵數字要背熟

| 數字 | 用途 |
|:---:|:----|
| $200/人/月 | 基本收費 |
| 25cr/月 | 基本每人 Credits |
| 2 個月免費試用 | 敲門磚 |
| 7 天上線 | 快速開通 |
| 15 分鐘 | Demo 長度 |

### 電話預約後

> 係咪同一個人可以跟，定係到時會轉介俾第二個同事？
>
> *（確認聯絡人冇變）*
